ATOMIC STRUCTURE

1. Definition of an Atom

An atom is the smallest particle of an element that retains the properties of that element. It consists of three main subatomic particles:

  • Protons (p⁺) – Positively charged, found in the nucleus.
  • Neutrons (n⁰) – Neutral (no charge), also in the nucleus.
  • Electrons (e⁻) – Negatively charged, orbit around the nucleus in energy levels (shells).

2. Structure of the Atom

  • Nucleus – Dense center containing protons and neutrons.
  • Electron Shells (Energy Levels) – Electrons revolve around the nucleus in these levels.

Maximum electrons in a shell: 2n² (where n = shell number, e.g., 1st, 2nd...)

3. Subatomic Particles

Particle Symbol Charge Mass (amu) Location
Proton p⁺ +1 1 Nucleus
Neutron n⁰ 0 1 Nucleus
Electron e⁻ -1 ~0 (1/1836) Electron cloud

4. Atomic Number and Mass Number

  • Atomic Number (Z): Number of protons in an atom.
  • Also equals the number of electrons in a neutral atom.
  • Mass Number (A): Total number of protons and neutrons.
  • A = Z + number of neutrons

5. Isotopes

Isotopes are atoms of the same element with the same number of protons but different number of neutrons.

Example:
Carbon-12 and Carbon-14 → both have 6 protons, but different mass numbers.

6. Electronic Configuration

Arrangement of electrons in shells around the nucleus.
Follow the rule: 2, 8, 8, 18...

Example:

  • Hydrogen (1e⁻) → 1
  • Carbon (6e⁻) → 2, 4
  • Sodium (11e⁻) → 2, 8, 1

7. Models of the Atom

Model Scientist Description
Solid Sphere Model John Dalton Atom as solid indivisible sphere
Plum Pudding Model J.J. Thomson Electrons in a positive "soup"
Nuclear Model Ernest Rutherford Atom mostly empty, dense nucleus
Bohr Model Niels Bohr Electrons in fixed orbits (energy levels)
Quantum Model Schrödinger etc. Electron cloud model (most accurate)

8. Importance of Atomic Structure

  • Basis of chemical bonding
  • Determines element behavior
  • Explains the periodic table
  • Essential in nuclear chemistry and quantum physics

Summary

  • Atoms have protons, neutrons, and electrons.
  • Atomic number = protons = electrons
  • Mass number = protons + neutrons
  • Electrons occupy energy levels
  • Isotopes differ in neutrons
  • Atomic structure explains reactivity and periodic trends.
